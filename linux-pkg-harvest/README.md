# Linux Package Harvest

A Python tool for harvesting package information from various Linux distribution repositories.

## Features

- Support for RPM-based distributions (OpenEuler, Fedora)
- Support for DEB-based distributions (Ubuntu, Debian)
- Package dependency analysis
- LibYear calculation for dependency freshness
 
## Project Structure

- `src/repoharvester/core/` - Core modules for data collection and processing
- `src/repoharvester/collectors/` - Collectors for different Linux distributions
- `src/repoharvester/detectors/` - Version detection modules
- `src/repoharvester/utils/` - Utility functions
- `configs/` - Configuration files
- `scripts/` - Execution scripts
- `tests/` - Test cases
